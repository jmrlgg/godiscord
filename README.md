# GO Discord
A community driven golang Discord bot! 


1. Create config.json

```
{
    "Token": "Discord Token",
    "BotPrefix": "Bot Prefix",
    "ClientID": "Reddit Client ID",
    "ClientSecret": "Reddit Client Secret",
    "RedditUser": "Reddit User",
    "RedditPass": "Reddit Pass"

}
```
